# VirtBMC - a modular ipmi server with pluggable backeds

VirtBMC is a complete rewrite of [VirtualBMC](https://docs.openstack.org/virtualbmc/latest/) project.

> [!NOTE]  
> Currently only individual backends are supported, interface similar to VirtualBMC is till in the works

## Backends Supported
- [OpenStack](/virtbmc_openstack/)
